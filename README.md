# Sales Data Analysis and Aggregation using Python

## Project Overview

This project focuses on data manipulation and aggregation using a sales dataset for April 2019. The objective was to clean the data, perform calculations, and analyze sales patterns to identify top-performing products and cities.

## What I Did

* Loaded the sales dataset into a Pandas DataFrame
* Extracted **City** and **State** information from the Purchase Address column
* Calculated total sales using Quantity Ordered and Price Each
* Identified the top 5 products based on total sales value
* Determined the city with the highest sales in April 2019
* Aggregated sales data by date to analyze daily sales trends
* Created visualizations to better understand sales performance

## How I Did It

* Used Python in Jupyter Notebook / Google Colab
* Used the **Pandas** library for data cleaning and aggregation
* Created a new column **Total Sales** using mathematical operations
* Applied grouping and sorting functions to analyze product and city performance
* Used **Matplotlib** to plot sales trends over time

## Result

* Successfully cleaned and processed the sales dataset
* Identified the top 5 best-selling products by total sales
* Determined the city that generated the highest revenue
* Observed daily sales trends throughout April 2019
* Demonstrated practical use of data manipulation and aggregation techniques in Python
